# NUS-TTSH-NHG SARS-CoV-2-Paxlovid Study 

This repository contains code and supporting materials for the study:
## A Framework to Evaluate Physiologic Feasibility of Repurpose Drugs across Populations for the Next Pandemic

## Overview

This project implements an integrated modelling framework combining:
- Physiologically Based Pharmacokinetics (PBPK)
- Pharmacodynamics (PD)
- Viral Dynamics (VD)
to evaluate treatment timing, dosing, duration, and adherence for nirmatrelvir–ritonavir.

## Repository Structure

- `code/` — Analysis and simulation scripts
- `data/` — Example or processed datasets (no raw clinical data)
- `results/` — Generated figures and tables
- `environment/` — Reproducibility information

## Data Availability

Due to data governance restrictions, raw clinical datasets are not publicly available.  
Where possible, processed outputs or example datasets are provided.

## Reproducibility

To reproduce the analysis:
1. Install required R packages (see `environment/`)
2. Run scripts in `code/` sequentially
3. Outputs will be generated in `results/`
